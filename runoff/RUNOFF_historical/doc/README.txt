This directory contains documentation related to the `RUNOFF' text
processor language.

`groff-man-pages':
	These are the actual man pages of the `groff' system that are
	related to historical `RUNOFF'.  You can get `groff' from
	<http://www.gnu.org/software/groff>.  You can get the most
	actual source files of the `groff' `RUNOFF' man-pages from
 	<http://cvs.savannah.gnu.org/viewvc/groff/contrib/RUNOFF/?root=groff>.
