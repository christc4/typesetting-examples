This directory contains the files related to `RUNOFF' in the groff system.

You can view these man-pages by

	$ man -l <file>
	
in text mode or by

	$ groffer <file> &
	
in graphical mode (PDF by default).

`<file>' is the file name of a man-page.  These are here:


runoff.7
	This is the groff documentation about the history of the
	`RUNOFF' language.


groff_filenames.7
	This documents the file name extensions of `RUNOFF', `roff'
	and `groff'.
