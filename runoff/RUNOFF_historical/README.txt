This `git' archive contains available information of the historical
`RUNOFF'.  This was the first text processor language, created
in the 1960s at MIT, being used and expanded up to the 1980s.
This archive contains 2 subdirectories:

`RUNOFF-files':
	These are the available original files written in the
	`RUNOFF' text language.
	
`doc':
	This directory contains the historical and actual doumentation
	files related to `RUNOFF'.

`hardware+OS':
	This will contain information and emulators for the computers
	of the 1960s and later on where `RUNOFF' was created and used.
