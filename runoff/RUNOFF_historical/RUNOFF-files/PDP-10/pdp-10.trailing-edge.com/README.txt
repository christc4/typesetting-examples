The wesite <pdp-10.trailing-edge.com> is qwuite large.  So we extracted
the files written in the `RUNOFF' language and some processed text
formats.


The `RUNOFF' source files are in the following directories:

RUNOFF_RNH: `RUNOFF' help files with extension `.rnh'
RUNOFF_RNO: normal `RUNOFF' files with extension `.rno'
RUNOFF_RUN: normal `RUNOFF' files with extension `.run'


The processed files are:

text_DOC: normal text files with extension `.doc'
text_MEM: normal text files with extension `.mem'
text_HLP: help files in text style with extension `.hlp'
text_PDF: modern files with extension `.pdf'
