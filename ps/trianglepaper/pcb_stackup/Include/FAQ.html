A. 1.0    How do I change the PCB Stackup in the picture?


Edit the pcb_stackup.eps file with any text editor (e.g. vi, gedit, gvim, emacs, etc.) At the top of the file is an array called layer_stackup which you edit to create your stackup. 
%   [(Layer Name)    (Layer#) (Cu Weight) (Thickness) (GrayScale) -> 1.0 White, 0.0 Black]
/layer_stackup [
    [(Top)              (1)     (0.5)        ()        (0.9)]
    [(02_GND_01)        (2)     (1.0)        ()        (0.6)]
    [(03_SIG_01)        (3)     (1.0)        ()        (0.8)]
    [(04_SIG_02)        (4)     (1.0)        ()        (0.8)]
    [(05_PWR_01)        (5)     (1.0)        ()        (0.4)]
    [(Bottom)           (6)     (0.5)        ()        (0.9)]
] def

A. 1.1    What if I don't know what the layer name/layer #/copper weight/thickness/grayscale of a layer is?


If you want to omit a layer variable, just substitue a null string () for it in the array.

A. 1.2    How do I view the PCB Stackup?


You can send the pcb_stackup.eps file to a PostScript printer ($ lp pcb_stackup.ps), or view it with GhostScript ($ /usr/bin/gs ) or Okular ($/usr/bin/okular ) or Evince ($ /usr/bin/evince ), or Gnome Ghostview ( /usr/bin/ggv ).

A. 1.3    How do I change the title of the PCB Stackup?


Edit the pcb_stackup.eps file with any text editor (e.g. vi, gedit, gvim, emacs, xemacs etc.) At the top of the file is a string called fab_name which you edit to change the title. The fab_name printed at the top of the page is scaled to fit the page no matter how long this title string is, so don't worry if you want to use a long descriptive name. 
/fab_name (Project_Name Board_Name PCB Stackup 271-XXXX-YY rev ZZ) def

A. 1.4    How do I use the PCB Stackup in another document?


You should be able to directly import the pcb_stackup.eps file into your document in its native EPS format. If you really need to convert it to a different graphic format, here's how to do so with Imagemagick:

Convert to a GIF w/ transparent background using Imagemagick: 

$ /usr/bin/convert pcb_stackup.eps pcb_stackup.gif

Convert to a PNG w/ white background using Imagemagick: 

$ /usr/bin/convert -flatten pcb_stackup.eps pcb_stackup.gif
