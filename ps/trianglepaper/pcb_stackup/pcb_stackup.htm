PCB Stackup

An Encapsulated PostScript Program to Display, Share, and Document PCB Stackups

Motivation

pcb_stackup.eps is a program that can make simple stackup drawings (see the below example images.) I'd been kicking around for some time the idea of making a way to quickly and unambiguously describe PCB stackups after I was unsatisfied with both ASCII art stackups and HTML tables, and additionally thought reusing External Manufacturer (EM) Microsoft Excel board fabrication spreadsheets to be overkill for most internal discussion and documentation purposes.

Methodology

A draft stackup is created by examining the requirements for a PCB. These requirements are driven by how many layers are needed to power distribution, signal routing, and mechanical thickness constraints (overall mechanical fit, thickness required to accommodate press fit pins, etc.) A draft stackup is created by using thicknesses gleaned from previous stackups and/or standard thicknesses of foil clad core laminate layers and the prepreg material used to both make and join them.

Mechanics

The stackups are generated by a PostScript program that is written in a format (Encapsulated PostScript or EPS) that allows it to be viewed or printed directly, or imported by other applications such as word processors, spreadsheets, etc. If a user wants to change the stackup, all they need to do is with any text editor, edit the layer_stackup array in the source code of the pcb_stackup.eps file:

The other smaller PCB stackups without layer thicknesses displayed above are also included in the source file, but commented out, for your reference.

Viewing

Okular (/usr/bin/okular) lets you view EPS files on your screen. The GNU Image Manipulation Program ( /usr/bin/gimp ) and the ImageMagick display program ( /usr/bin/display ) are also tools that you can use to view EPS files.

It's a concept some have difficulty comprehending, but you can send this EPS file straight to a PostScript printer (virtually all our shared printers are PostScript printers), and it will print out the picture of the PCB stackup, not the source code text. If you actually want to print out the source code text, you have to use a pretty printer like enscript in this fashion: enscript -G --pretty-print=postscript --font=Courier9 --header='Your Name' pcb_stackup.eps I've created a PDF of the program's source code pretty print output for your convenience.

Sharing

This EPS file can be directly imported into a word processing program as an .eps file or distilled into a standalone PDF with ps2pdf ( /usr/bin/ps2pdf ). The ImageMagick convert program ( /usr/bin/convert etc. ) can be used to view and convert EPS files to other formats (e.g. PNG, JPEG, GIF). Information for doing so is contained in the source code of the pcb_stackup.eps file.
